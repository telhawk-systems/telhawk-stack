# Helper Scripts

Purpose: Brief descriptions of helper scripts available in `scripts/`.

- `scripts/install-hooks.sh` — Installs recommended Git hooks.
- `scripts/pre-push.sh` — Formatting, lint, vet, and tests; mirrors CI checks.
- `scripts/curl.sh` — Handy curl wrappers for local endpoints.
- `scripts/script_exec.sh` — Utility for running scripts with standard env.

See `docs/CI_DEVELOPMENT.md` for CI details.
